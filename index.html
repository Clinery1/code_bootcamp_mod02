Structural Editor

The purpose of this editor is to edit s-expressions. It uses structural editing to edit the content of the s-expressions and display them in an easy-to-read format. Since this is made to edit s-expressions, it uses simple text files as input and output, but it formats the files upon output for readability. I am using the editor (aliased to the sedit command to edit HTSX files (another project of mine) for this assignment.

HTSX language

The HTSX language is basically HTML, but it instead uses s-expressions to represent the HTML tags. Since it almost a 1:1 replica of HTML, it is a simple conversion from s-expressions to HTML
