Structural Editor

The purpose of this editor is to edit s-expressions.It uses structural editing to edit the content of the s-expressions and display them in an easy-to-read format.Since this is made to edit s-expressions, it uses simple text files as input and output, but it formats the files upon output for readability.I am using the editor (aliased to theseditcommand to edit HTSX files (another project of mine) for this assignment.

HTSX language

The HTSX language is basically HTML, but it instead uses s-expressions to represent the HTML tags.Since it almost a 1:1 replica of HTML, it is a simple conversion from s-expressions to HTML

Docbuilder

This project was built entirely so I could build myself a good looking Resume without using anything microsoft.Due to the simplicity of the language, it has a lot of issues and so is getting a rewrite as an s-expression-based language like HTSX.

Dotfiles

Dotfiles are an essential part of one's configuration, so it is always a good idea to not only back them up and log changes, but to share them so others can take inspiration.Without other people's dotfiles mine would never be the way they are now, and I wouldn't be able to do the things I can today.Although it may seem useless to track changes to configuration files, tracking the changes makes it trivial to revert a change that broke something, or if you deleted something that you didn't want to.

TVARA

TVARA is an application made by Christian, Matthew, Alex, and I. It tracks the ingredients you have in your pantry and allows you to search recipes. It also allows you to update your pantry with the press of a button after cooking a recipe.TVARA also includes a save feature that allows you to save favorited recipes in your browser for easy retrieval later.
